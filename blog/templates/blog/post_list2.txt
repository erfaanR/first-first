class="_13fihbu">Each of the boids (bird-oid objects) obeys three simple rules:</p><h3 class="_1461p8n">1. Coherence</h3><p class="_13fihbu">Each boid flies towards the the other boids. But they don't just immediately fly directly at each other. They gradually steer towards each other at a rate that you can adjust with the "coherence" slider.</p><h3 class="_1461p8n">2. Separation</h3><p class="_13fihbu">Each boid also tries to avoid running into the other boids. If it gets too close to another boid it will steer away from it. You can control how quickly it steers with the "separation" slider.</p><h3 class="_1461p8n">3. Alignment</h3><p class="_13fihbu">Finally, each boid tries to match the vector (speed and direction) of the other boids around it. Again, you can control how quickly they try to match vectors using the "coherence" slider.</p><h2 class="_1sx1guo">Visual range</h2><p class="_13fihbu">There are a ton of ways to extend this simple model to better simulate the behavior of different animals. An example I showed in the video is to limit the "visual range" of each boid. Real animals can't see the entire flock; they can only see the other animals around them. By adjusting the visual range slider, you can adjust how far each boid can "see"—that is which other boids it considers when applying the three rules above.</p><h2 class="_1sx1guo">What else can I do?</h2><p class="_13fihbu">The source code from the video is available on <a class="_1iw9e0l" to="https://github.com/beneater/boids" target="_blank" href="https://github.com/beneater/boids" dideo-checked="true">GitHub</a>. If you want to go beyond what you can do by playing with the sliders on this page, I recommend grabbing the code and changing stuff to see what happens. All you need is a text editor and a web browser.</p><p class="_13fihbu">There are lots of features you could try adding to the code yourself:<ul class="_1q7xlyy"><li>Add a predator that the boids try to avoid that scatters the flock if it gets too close.</li><li>Add a strong wind or current to see what effect it has on the flock.</li><li>Add "perching" behavior.
